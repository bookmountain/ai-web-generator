# AI Web Generator - Frontend

Frontend for an AI code generator built with Vue 3, TypeScript, and Ant Design Vue.

## Features

- Create apps from prompts
- Chat with AI and preview results in real time
- Edit, delete, and view your apps
- Deploy apps to the cloud
- Browse your apps and featured apps
- Admin tools for managing any app and user

## Main Pages

- `/` — Home
- `/app/chat/:id` — App chat
- `/admin/appManage` — App management
- `/app/edit/:id` — App editing

## Tech Stack

- Vue 3
- TypeScript
- Ant Design Vue
- Vue Router 4
- Pinia
- Vite
- Axios
- Day.js
